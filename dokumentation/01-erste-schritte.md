# Erste Schritte mit Kaderblick

Willkommen bei **Kaderblick** — der Web-App für euren Fußballverein! Hier erfahrt ihr, wie ihr schnell startet und die wichtigsten Funktionen findet.

---

## Was ist Kaderblick?

Kaderblick ist eine **Web-Anwendung** für Fußballvereine, die alles an einem Ort vereint:

- **Termine & Kalender** — Trainings, Spiele und Turniere im Überblick
- **Teilnahme** — Zu- und Absagen per Klick
- **Spielverwaltung** — Ergebnisse, Spielereignisse, Video-Analyse
- **Aufgaben** — Wer bringt die Getränke mit? Wer wäscht die Trikots?
- **Fahrgemeinschaften** — Mitfahrgelegenheiten organisieren
- **Nachrichten & News** — Kommunikation im Team
- **Umfragen** — Meinungen einholen
- **Dashboard** — Eure persönliche Startseite mit Widgets
- **Berichte & Statistiken** — Tore, Karten, Trainingsbeteiligung und mehr
- **Formationen** — Taktische Aufstellungen planen
- **XP-System** — Punkte sammeln für aktive Teilnahme

Kaderblick läuft im **Browser** (Chrome, Firefox, Safari, Edge) und auf dem **Handy** — ihr braucht keine App aus dem Store zu installieren. Einfach die URL aufrufen, einloggen und loslegen.

---

## Registrierung

Um Kaderblick zu nutzen, braucht ihr einen **Account**. Es gibt zwei Wege:

### 1. E-Mail-Registrierung

1. Geht auf die Registrierungsseite
2. Gebt **Vorname**, **Nachname**, **E-Mail** und **Passwort** ein
3. Klickt auf **„Registrieren"**
4. Ihr erhaltet eine **Bestätigungs-Mail** — klickt auf den Link darin
5. Nach der Bestätigung könnt ihr euch einloggen

### 2. Google-Anmeldung

1. Klickt auf **„Mit Google anmelden"**
2. Wählt euer Google-Konto aus
3. Fertig — euer Account wird automatisch erstellt

> 💡 Die Google-Anmeldung ist besonders praktisch, da ihr euch kein zusätzliches Passwort merken müsst.

---

## Anmeldung (Login)

1. Geht auf die Login-Seite
2. Gebt eure **E-Mail** und euer **Passwort** ein
3. Klickt auf **„Anmelden"**

Oder: Klickt auf **„Mit Google anmelden"**, falls ihr euch mit Google registriert habt.

### Passwort vergessen?

1. Klickt auf **„Passwort vergessen?"** auf der Login-Seite
2. Gebt eure **E-Mail-Adresse** ein
3. Ihr erhaltet einen **Reset-Link** per Mail
4. Klickt auf den Link und setzt ein neues Passwort

## Die Benutzeroberfläche

### Navigation

Am oberen Rand findet ihr die **Hauptnavigation** mit den wichtigsten Bereichen:

| Menüpunkt | Was findet ihr dort? |
|-----------|---------------------|
| **Dashboard** | Eure persönliche Startseite mit Widgets |
| **Kalender** | Alle Termine (Trainings, Spiele, Turniere) |
| **Spiele** | Spielübersicht mit Ergebnissen und Ereignissen |
| **Turniere** | Turnierverwaltung mit Bracket und Gruppenphase |
| **News** | Vereins- und Team-Nachrichten |
| **Nachrichten** | Persönliche Nachrichten (Posteingang) |
| **Aufgaben** | Wiederkehrende Pflichten und To-Dos |
| **Spieler** | Kader-Übersicht |
| **Trainer** | Trainer-Verwaltung |
| **Formationen** | Taktische Aufstellungen |
| **Berichte** | Statistiken und Auswertungen |
| **Umfragen** | Abstimmungen und Meinungsabfragen |

### Auf dem Handy

Auf kleineren Bildschirmen wird die Navigation in ein **Hamburger-Menü** (☰) zusammengefasst. Tippt darauf, um alle Menüpunkte zu sehen.

### Benachrichtigungs-Glocke

Oben rechts seht ihr eine **Glocke** 🔔:

- **Ausgefüllte Glocke** mit **Zahl** → Ihr habt ungelesene Benachrichtigungen
- **Umriss-Glocke** → Alles gelesen, keine neuen Nachrichten
- Klick → **Benachrichtigungs-Fenster** öffnet sich (mehr in [15-benachrichtigungen.md](15-benachrichtigungen.md))

### Profilbild

Oben rechts seht ihr euer **Profilbild** (oder einen Platzhalter). Klickt darauf für:

- **Profil bearbeiten** — Persönliche Daten, Profilbild, Passwort
- **Abmelden**

## Erste Einrichtung

### 1. Profilbild hochladen

Ein gutes Profilbild hilft allen, euch zu erkennen — in Nachrichten, Teilnehmerlisten und Kommentaren.

1. Klickt auf euer Profilbild oben rechts → **„Profil"**
2. Ladet ein Bild hoch (Drag & Drop oder Klick)
3. Schneidet es zu (runder Ausschnitt)
4. Speichern

Mehr Details in [19-profil.md](19-profil.md).

### 2. Push-Benachrichtigungen aktivieren

Damit ihr keine wichtigen Termine oder Nachrichten verpasst:

1. Geht in euer **Profil**
2. Scrollt zu **„Push-Benachrichtigungen"**
3. Klickt auf **„Push aktivieren"**
4. Erlaubt die Benachrichtigungen im Browser

Mehr Details in [15-benachrichtigungen.md](15-benachrichtigungen.md) und [19-profil.md](19-profil.md).

### 3. Nutzer-Verknüpfung herstellen

Wenn ihr ein **Spieler** oder **Elternteil** seid, verknüpft euren Account mit dem Spieler-/Trainer-Profil:

1. Profil → **Verknüpfungen** (🔗)
2. „Neue Verknüpfung" → Typ wählen (Spieler/Trainer) → Person auswählen
3. Die Verknüpfung wird von einem Admin bestätigt

Mehr Details in [19-profil.md](19-profil.md).

### 4. Dashboard einrichten

Richtet euer Dashboard mit den für euch wichtigsten Widgets ein:

1. Geht auf das **Dashboard**
2. Klickt auf **„Widget hinzufügen"**
3. Wählt z. B. „Nächste Termine", „News", „Nachrichten"
4. Passt Größe und Reihenfolge an

Mehr Details in [16-dashboard.md](16-dashboard.md).

## Rollen und Berechtigungen

In Kaderblick gibt es verschiedene **Rollen**, die bestimmen, was ihr sehen und tun könnt:

| Rolle | Was kann ich? |
|-------|--------------|
| **Spieler / Mitglied** | Termine sehen, zu-/absagen, Nachrichten lesen/schreiben, Aufgaben erledigen, eigenes Profil bearbeiten |
| **Elternteil** | Wie Spieler, plus Einsicht in die Daten des verknüpften Kindes |
| **Trainer** | Alles von Spieler, plus: Spiele anlegen, Aufgaben erstellen, Videos verwalten, Berichte erstellen, Formationen planen |
| **Administrator** | Voller Zugriff — Nutzer verwalten, Verknüpfungen bestätigen, Stammdaten pflegen, Feedback bearbeiten |

> ℹ️ Manche Funktionen (z. B. Spiele anlegen, Aufgaben erstellen) sind nur für Trainer und Administratoren sichtbar. Als Spieler oder Elternteil seht ihr die Ergebnisse, könnt aber nichts anlegen.

## Hilfe und Feedback

### Feedback-Button

Auf jeder Seite gibt es einen **Feedback-Button** (unten rechts, schwebendes Symbol):

1. Klickt darauf
2. Wählt den **Typ**: Fehler (Bug), Verbesserungsvorschlag (Feature), Frage oder Sonstiges
3. Schreibt eure **Nachricht**
4. Optional: Ein **Screenshot** der aktuellen Seite wird automatisch erstellt und mitgeschickt
5. Absenden

Das Feedback wird direkt an die Administratoren weitergeleitet. Ihr müsst nichts per E-Mail schicken oder WhatsApp schreiben.

## Anwendung auf dem Handy nutzen

Kaderblick ist eine **Progressive Web App (PWA)** — ihr könnt sie wie eine echte App auf eurem Handy-Homescreen installieren:

### Installation auf dem Handy

#### Android (Chrome)
1. Öffnet Kaderblick im Chrome-Browser
2. Tippt auf die **drei Punkte** (⋮) oben rechts
3. Wählt **„Zum Startbildschirm hinzufügen"**
4. Bestätigt → Kaderblick erscheint als App-Symbol auf eurem Homescreen

#### iPhone (Safari)
1. Öffnet Kaderblick in Safari
2. Tippt auf das **Teilen-Symbol** (↑) unten
3. Scrollt runter → **„Zum Home-Bildschirm"**
4. Bestätigt → Kaderblick ist jetzt wie eine App auf eurem iPhone

> 💡 Nach der Installation öffnet sich Kaderblick im **Vollbild-Modus** — ohne Browser-Adressleiste, genau wie eine richtige App!
