# Nachrichten (Chat & Messaging)

Mit der Nachrichten-Funktion kommuniziert ihr direkt mit einzelnen Personen oder ganzen Gruppen — ohne WhatsApp oder externe Messenger. Alles bleibt innerhalb von Kaderblick.

---

## Nachrichten öffnen

In der oberen Navigationsleiste seht ihr das **Briefumschlag-Symbol** (✉️). Es zeigt euren Posteingang an:

- Eine **rote Zahl** (Badge) zeigt, wie viele **ungelesene Nachrichten** ihr habt
- Klickt auf das Symbol → es öffnet sich das **Nachrichten-Fenster**

> 💡 Das Briefumschlag-Symbol arbeitet **unabhängig** von der Benachrichtigungsglocke (🔔). Die Glocke zeigt alle Benachrichtigungstypen, der Briefumschlag zeigt nur Nachrichten.

---

## Der Posteingang

Das Nachrichten-Fenster ist in zwei Bereiche aufgeteilt:

### Linke Seitenleiste

- **„Posteingang"** — mit einer Zahl, die eure ungelesenen Nachrichten zeigt
- **Nachrichtengruppen** — eine Liste eurer Gruppen-Chats (z. B. „U13 Eltern", „Trainerteam")

Klickt auf eine Gruppe, um deren Nachrichten zu sehen.

### Rechter Hauptbereich — Nachrichtenliste

Eine Tabelle mit allen Nachrichten, neueste oben:

| Spalte | Beschreibung |
|--------|-------------|
| **Symbol** | 📧 Geschlossener Brief = ungelesen, 📨 Offener Brief = gelesen |
| **Betreff** | Der Betreff der Nachricht — **fett** bei ungelesenen Nachrichten |
| **Absender** | Wer hat die Nachricht geschickt? |
| **Datum** | Wann wurde sie gesendet? |

Klickt auf eine Nachricht in der Liste → es öffnet sich die **Detailansicht**.

---

## Eine Nachricht lesen

In der Detailansicht seht ihr:

- **Absender** — Name des Absenders
- **Datum** — Wann wurde die Nachricht gesendet?
- **Empfänger** — An wen wurde die Nachricht geschickt?
- **Vollständiger Inhalt** — der komplette Nachrichtentext

### Antworten

Unter der Nachricht gibt es den Button **„Antworten"**:

1. Klickt auf **„Antworten"**
2. Es öffnet sich das Schreibfeld — der Betreff wird automatisch mit **„Re: [Originalbetreff]"** vorausgefüllt
3. Der Originaltext wird als **Zitat** unter eurer Antwort eingefügt
4. Schreibt eure Antwort und klickt auf **„Senden"**

---

## Eine neue Nachricht schreiben

1. Im Nachrichten-Fenster klickt auf **„Neue Nachricht"** (Stift-Symbol ✏️)
2. Es öffnet sich das Schreibfenster:

| Feld | Beschreibung |
|------|-------------|
| **Empfänger** | Beginnt zu tippen — es werden passende Namen vorgeschlagen (Autocomplete). Ihr könnt **mehrere Empfänger** auswählen. Angezeigt werden Name und E-Mail-Adresse. |
| **Gruppe** | Alternativ: Wählt eine **Nachrichtengruppe** aus dem Dropdown — alle Gruppenmitglieder werden automatisch als Empfänger eingetragen |
| **Betreff** | Kurzer Betreff, der in der Übersicht angezeigt wird |
| **Nachricht** | Der eigentliche Nachrichtentext (beliebig lang) |

3. Klickt auf **„Senden"** — die Nachricht wird sofort zugestellt

### Empfänger auswählen — im Detail

Das Empfängerfeld funktioniert als **Suchfeld mit Mehrfachauswahl**:

- Tippt den Anfang eines Namens → passende Nutzer werden als Vorschläge angezeigt
- Klickt auf einen Namen → er wird als **Chip** (kleines Kärtchen) im Feld angezeigt
- Wiederholt das für weitere Empfänger
- Klickt auf das **X** an einem Chip, um den Empfänger wieder zu entfernen

---

## Nachrichtengruppen

Gruppen sind wie Verteilerlisten — ihr schickt eine Nachricht an die Gruppe und **alle Mitglieder** erhalten sie.

### Eine Gruppe erstellen

Gruppen werden von Trainern oder Administratoren erstellt:

1. Im Nachrichten-Fenster den Bereich **Nachrichtengruppen** in der Seitenleiste aufklappen
2. Auf **„Neue Gruppe"** klicken
3. **Gruppenname** eingeben (z. B. „U13 Eltern", „Trainerteam A-Jugend")
4. **Mitglieder** hinzufügen — über das gleiche Suchfeld wie bei Empfängern
5. **Speichern** — die Gruppe erscheint in der Seitenleiste

### Eine Nachricht an eine Gruppe senden

1. Beim Schreiben einer neuen Nachricht: Im Feld **„Gruppe"** die gewünschte Gruppe auswählen
2. Die Mitglieder werden automatisch als Empfänger eingetragen
3. Betreff und Nachricht eingeben, **„Senden"** klicken

### Gruppen verwalten

- Bearbeiten: Mitglieder hinzufügen oder entfernen
- Löschen: Die Gruppe wird aufgelöst — bisherige Nachrichten bleiben erhalten

---

## Gelesen / Ungelesen

Kaderblick merkt sich, welche Nachrichten ihr gelesen habt:

- **Ungelesene Nachrichten** → geschlossenes Briefsymbol 📧, **fetter Betreff**, erscheinen im roten Badge-Zähler
- **Gelesene Nachrichten** → offenes Briefsymbol 📨, normaler Betreff

Eine Nachricht wird automatisch als „gelesen" markiert, sobald ihr sie **öffnet**.

---

## Benachrichtigungen bei neuen Nachrichten

Wenn euch jemand eine Nachricht schickt, werdet ihr auf **drei Wegen** informiert:

1. **Briefumschlag-Badge** (✉️) in der Navigation — die rote Zahl steigt
2. **Glocken-Benachrichtigung** (🔔) — eine Benachrichtigung vom Typ „Nachricht" erscheint
3. **Push-Benachrichtigung** (falls aktiviert) — euer Gerät zeigt eine Benachrichtigung, auch wenn die App geschlossen ist

## Praxisbeispiele

| Situation | Empfänger | Beispiel-Betreff |
|-----------|----------|-----------------|
| Absprache mit einem Mitspieler | Einzelperson | „Mitfahrgelegenheit Samstag?" |
| Trainer informiert Eltern | Gruppe „U13 Eltern" | „Geänderter Trainingsort" |
| Trainer an einen Spieler | Einzelperson | „Deine Aufstellung für Samstag" |
| Elternteil an Trainer | Einzelperson | „Tim kann leider nicht kommen" |
| Trainerteam-Abstimmung | Gruppe „Trainerteam" | „Vorbesprechung Turnierwochenende" |

## Häufige Fragen

### Kann ich eine gesendete Nachricht zurücknehmen?
Nein — gesendete Nachrichten können nicht zurückgenommen oder gelöscht werden. Überlegt kurz, bevor ihr „Senden" klickt.

### Sieht der Absender, ob ich die Nachricht gelesen habe?
Der Absender sieht, dass die Nachricht zugestellt wurde. Ob ihr sie gelesen habt, ist für den Absender nicht direkt sichtbar.

### Kann ich Dateien oder Bilder anhängen?
Aktuell unterstützt das Nachrichtensystem reinen Text. Für Bilder oder Dokumente könnt ihr Links teilen.

### Wie viele Mitglieder kann eine Nachrichtengruppe haben?
Es gibt keine technische Begrenzung. Achtet aber darauf, dass die Gruppen übersichtlich bleiben — lieber mehrere kleinere Gruppen als eine riesige.

### Bekomme ich Nachrichten auch, wenn ich offline bin?
Die Nachricht wird gespeichert und wartet auf euch. Sobald ihr die App öffnet, seht ihr sie in eurem Posteingang. Falls Push aktiviert ist, werdet ihr auch über das Gerät informiert.

### Was ist der Unterschied zwischen Nachrichten und News?
**Nachrichten** sind direkte Kommunikation (wie E-Mails) — zwischen Personen, mit Antwortmöglichkeit. **News** sind Veröffentlichungen (wie ein Aushang) — von Trainern/Admins an eine Zielgruppe, ohne Antwortmöglichkeit.
